自建RSS服务

背景

本文简单记录搭建 rss 服务(自用)过程，之前追踪技术博客都是收藏到浏览器书签，此方法无法知道博客合适更新，机缘巧合之下了解到了 rss，因此动手搭建自己的 rss 服务，推荐先阅读此文对 rss 有一个初步了解，以决定是否需要使用 rss。

在浏览器中打开 http://127.0.0.1:1200/ 即可访问。为了安全，建议添加访问控制，在项目根目录新建一个 .env 文件，每行以 NAME=VALUE 格式添加环境变量：

  • CACHE_TYPE 指定缓存方式，这里为 redis
  • CACHE_EXPIRE 指定缓存过期时间
  • REDIS_URL 指定 redis 连接地址
  • ACCESS_KEY 指定访问 RSSHub 服务所提供的 key，需要在 url 中提供
  • LISTEN_INADDR_ANY 是否允许公网连接，默认 1
  • PORT 监听端口，默认为 1200

更多配置选项，请参阅官方手册。这里指定监听地址为本地地址，因此也需要使用 nginx 进行反向代理，具体请参考前文。
